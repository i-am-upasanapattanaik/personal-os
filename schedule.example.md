## Planning Notes for Agent

- **Default slot preference**: Use daytime slots first (morning/afternoon). Use evening slots only if daytime is unavailable or explicitly chosen.
- **Buffer time**: Always leave 15 min buffer between tasks for context switching.
- **P0 tasks**: Schedule during peak focus hours (first 2–3 hours of the day's slot).
- **Weekly nudge**: Compare completed tasks against P0/P1 goals at end of each week.
